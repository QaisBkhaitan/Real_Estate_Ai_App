# Real Estate AI App

AI-powered real estate assistant that helps users search, explore, and understand property listings using intelligent chat and retrieval (RAG).
---
##  Features

-  AI Chat Assistant for property queries  
-  Property listing and filtering  
-  Smart search using Retrieval-Augmented Generation (RAG)  
-  FastAPI backend for API handling  
-  React frontend (Vite) for user interaction  
-  SQLite database for property storage  
-  WebSocket communication (real-time interaction)

---

## How It Works

1. User sends a question (e.g. *"Show me apartments in Ramallah under $100k"*)  
2. Backend retrieves relevant property data  
3. AI model processes the query using RAG  
4. Response is generated and returned to the frontend  

---

## 🛠️ Tech Stack

### Backend
- FastAPI  
- Python  
- SQLite  

### AI
- RAG (Retrieval-Augmented Generation)  
- Embeddings + Vector Search  
- LLM (Groq / Llama)

### Frontend
- React (Vite)  
- JavaScript  
- WebSocket API
